# Pasta de Fontes — Ovelin

## Fonte Defante (para Títulos)

Adicione os arquivos da fonte **Defante** nesta pasta com os nomes:

- `Defante.woff2` ← formato preferido (menor tamanho)
- `Defante.woff`  ← fallback
- `Defante.ttf`   ← fallback adicional

O CSS já está configurado para carregá-los automaticamente via `@font-face`.

> Se você tiver o arquivo `.ttf`, pode convertê-lo para `.woff2` gratuitamente em:
> https://cloudconvert.com/ttf-to-woff2

## Fontes do Google (carregadas automaticamente)

- **Comfortaa** → subtítulos, navegação, botões
- **Poppins**   → textos, parágrafos, formulários
